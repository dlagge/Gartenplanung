# Gartenplanung

## Bewegen der Gartenkarte
Ziehen: rechter Mausklick halten
Drehen: Shift + rechter Mausklick halten
Zoomen: Mausrad


## run project
npx vite


## installation

### node
npm init

### three.js
npm install --save three

### vite
npm install --save-dev vite

### air datepicker
npm i air-datepicker

### mysql
npm install mysql2
